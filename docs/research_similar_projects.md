## Key lesson

The best tools do not show every technical feature first.

They guide the creator through a visible workflow:

```text
story
boards or panels
images
text or lettering
review
export
backup or project file
```

## Strong patterns to copy

```text
simple start screen
one main action per screen
panel board as central object
right-side metadata or help panel
visible progress
export as a clear final step
advanced tools hidden until needed
```

## Strong patterns to avoid

```text
too many routes at once
technical labels as primary navigation
training tools before a first rough episode
export hidden behind production jargon
empty states without instructions
```

## Direct product decision

The Comic Factory should behave like a guided episode assistant.

Beginner path:

```text
Start
Episode Board
Make Images
Choose Images
Add Text
Export
Save Backup
```

Advanced path:

```text
References
Assets
Storage
Restore
Dataset
Training
Old Pages
```
