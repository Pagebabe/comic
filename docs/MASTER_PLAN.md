# Comic Factory Masterplan

## Oberstes Ziel

Eine Person ohne professionelle Animationserfahrung produziert zuerst eine gute, kurze und wiederholbare Szene. Die Software wächst erst aus bewiesenen Produktionsschritten.

## Meilensteine

1. M0 Bestand & Neustart
2. M1 Lebenszeichen
3. M2 Mini-Szene
4. M3 Character Lock
5. M4 Erste gute Folge
6. M5 Wiederholung
7. M6 Automationsanalyse
8. M7 Comic Factory V1

## Stop-Regeln

- Vor M2 keine neue Plattformarchitektur.
- Vor M3 keine dritte aktive Figur.
- Vor Animatic-Freigabe keine teuren Finalrenders.
- Kein Tool ohne konkrete Aufgabe.
- Kein LLM darf unkontrolliert schreiben oder ausführen.
