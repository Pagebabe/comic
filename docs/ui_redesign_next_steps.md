# UI Redesign Next Steps

Last updated: 2026-07-05

## Goal

Make the app simple for beginners.

## First change

The visible navigation should become:

```text
Start
Episode Board
Make Images
Choose Images
Add Text
Export
Save Backup
Advanced
```

## Page standard

Every production page needs:

```text
page title
short explanation
one primary action
progress state
next step
small help area
```

## Advanced tools

Move these behind Advanced:

```text
Storage
Restore
References
Dataset tools
Training tools
Legacy pages
```

## First implementation order

```text
1. Rename sidebar labels for beginners.
2. Hide legacy pages from default sidebar.
3. Add current next action card to Start.
4. Add right help card to Image Review.
5. Add right help card to Lettering.
6. Add locked states to Export.
7. Add Save Backup as final visible step.
```

## Design standard

```text
one main button per page
simple labels
advanced options collapsed
clear empty states
clear progress
clear next action
```
