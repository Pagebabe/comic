# UI Phase 1 Navigation

Last updated: 2026-07-05

## Goal

Make the sidebar easier for beginners.

## Change

Old groups were technical:

```text
Production
Assets
Training
Output
System
Legacy
```

New groups are task based:

```text
Episode Flow
Advanced Tools
Old Pages
```

## Beginner path

```text
Start
Episode Board
Make Images
Render Helper
Upload Images
Choose Images
Add Text
Export
Save Backup
```

## Advanced path

```text
Workspace Map
Generation Queue
QA Gate
Storage Safety
Restore Backup
Reference tools
Asset tools
Dataset tools
Training plan
```

## Rule

A beginner should first see the path to finish Episode 001.

Advanced tools stay available, but they no longer interrupt the main episode flow.

## Next UI phase

Simplify the Start page.

It should show:

```text
current episode
current panel
next action
progress
one continue button
```
