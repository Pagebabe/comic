# Beginner UI Principles

Last updated: 2026-07-05

## Goal

The Comic Factory must be easy for beginners.

Every screen must answer:

```text
Where am I?
What is this for?
What do I do next?
What is done?
What is missing?
When am I finished?
```

## Product focus

```text
Finish Episode 001 as a rough comic or rough video.
```

## Rules

```text
One main action per page.
Use plain language.
Show progress everywhere.
Show next step everywhere.
Explain confusing controls beside the control.
Hide advanced tools until needed.
Never show an empty page without telling the user what to do.
Never show a warning without saying how to fix it.
```

## Standard page layout

```text
left navigation
page header
main work area
right help panel
bottom save status
```

## Beginner path

```text
Start
Episode Board
Make Images
Choose Images
Add Text
Export
Save Backup
```

## Advanced path

```text
Storage
Restore
References
Training Prep
Legacy pages
```

Advanced tools should be collapsed by default until Episode 001 is finished.
