# Episode 001 Panel Production Cards

Last updated: 2026-07-05

## Purpose

Production sheet for the hand-test of:

```text
Ricco im Haus — Episode 001: Das Zimmer
```

This is not a new tooling feature.

Use it to generate rough panel images, pick finals and stop endless tweaking.

## Global style

```text
gritty adult satirical cartoon, run-down Berlin house, bold black outlines, grounded exaggerated faces, dirty muted colors, messy lived-in interiors, social satire mood, no speech bubbles, no text inside image
```

## Global acceptance rule

For each panel:

```text
Generate 2 to 4 variants.
Pick one final once the scene is readable.
Do not chase perfect faces.
Do not regenerate unless the story action is broken.
```

# Panel 1 — Ankunft

## Goal

Introduce Ricco, the house and Don Miau.

## Image target

Ricco stands in front of the run-down Berlin house with bags, backpack, sports bag and cheap music gear. Don Miau watches from an upper window.

## Must have

```text
Ricco with moving bags
old dirty house facade
Don Miau in window
arrival mood
```

## Reject if

```text
cat missing
house looks clean or expensive
Ricco looks rich
```

## Dialogue

```text
Ricco: Bruder, endlich was Eigenes.
```

# Panel 2 — Basti erscheint

## Goal

Introduce Basti as friendly but slippery landlord figure.

## Image target

Basti arrives clean and smug with e-bike detail, coffee cup, tote bag and round glasses. Ricco stands with bags in front of the dirty house.

## Must have

```text
Basti clean and moral-smiling
Ricco still moving in
clear contrast between Basti and building
```

## Reject if

```text
Basti looks like gangster or CEO
Ricco looks dominant
scene lacks house context
```

## Dialogue

```text
Basti: Ricco, schön, dass du da bist. Wichtig: Das hier ist kein Mietverhältnis.
Ricco: Noch besser. Dann muss ich ja keine Miete zahlen.
```

## Must have

```text

pallet mattress
cheap music setup
Don Miau on bed
Ricco disappointed
```

## Reject if
