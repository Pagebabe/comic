# UI Rules Short

Last updated: 2026-07-05

## Goal

The app must guide beginners through one complete episode.

## Rules

```text
Show status on every page.
Use simple language.
Keep one main action per page.
Keep advanced tools collapsed.
Explain important inputs near the input.
Show the next step on every work page.
Show what is done and what is missing.
Use confirmations for risky actions.
Use simple recovery text for problems.
```

## Beginner flow

```text
Start
Episode Board
Make Images
Choose Images
Add Text
Export
Save Backup
```

## Working rule

Do not add more UI power until this beginner flow is clear.
