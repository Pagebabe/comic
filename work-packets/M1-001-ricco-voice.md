# M1-001 · Ricco-Arbeitsstimme auswählen

## Sichtbares Ergebnis

Eine kurze WAV-Datei mit Riccos gesperrtem Testsatz, geeignet für den 3–5-Sekunden-Lebenszeichen-Test.

## Testsatz

`Bruder, endlich was Eigenes.`

## Anforderungen

- Deutsch verständlich
- sympathisch-chaotisch statt überdreht
- trockenes Timing
- keine dauerhafte Voice-Cloning-Entscheidung
- WAV, mono oder stereo, 48 kHz bevorzugt

## Vergleich

Höchstens drei Varianten vergleichen:

1. eigene Arbeitsaufnahme
2. vorhandene lokale TTS
3. genau ein Cloud-TTS-Anbieter

## Definition of Done

- genau eine Arbeitsstimme ausgewählt
- Audiodatei eindeutig benannt
- kurze Begründung dokumentiert
- keine neue Audio-Plattform gebaut
