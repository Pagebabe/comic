# Agent Rules

1. Aktive Linie ist M1 Lebenszeichen.
2. Keine Funktion ohne sichtbaren Nutzen für Test A.
3. GitHub ist die verbindliche Wahrheit.
4. Ein Arbeitspaket entspricht einem klaren Ziel.
5. Keine freie Shell aus Bot- oder LLM-Ausgaben.
6. Änderungen benötigen Tests, Beweise und Dokumentation.
7. Neue Figuren und Locations werden zuerst als Vorschlag behandelt.
8. Cloud- und Renderkosten benötigen Freigabe.
